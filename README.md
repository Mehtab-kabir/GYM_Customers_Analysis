# Gym Customers Analysis — Power BI Dashboard

An interactive **Power BI** dashboard that analyzes and visualizes gym customer
data, turning raw membership/activity records into insights about the customer
base.

## What this project does

This project provides a **people / customer data visualization** dashboard for a
gym, helping management understand who their members are and how they behave. It
visualizes key customer metrics to support data-driven decisions around
membership, retention, and marketing.

> The dashboard is built in Power BI Desktop. Specific visuals and KPIs are
> defined inside `GYM_Dashboard.pbix`; open it in Power BI Desktop to explore
> them interactively.

## Repository contents

```
GYM_Customers_Analysis/
├── GYM_Dashboard.pbix          # The Power BI dashboard
└── Dataset PBI Playlist.xlsx   # Source dataset
```

## How to use

1. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (Windows, free).
2. Download / clone this repository.
3. Open `GYM_Dashboard.pbix` in Power BI Desktop.
4. If prompted, point the data source at `Dataset PBI Playlist.xlsx` from this
   repository, then **Refresh** to load the data.
5. Interact with the slicers and visuals to explore the customer analysis.

## Tools used

- **Power BI Desktop** — dashboard and data modeling
- **Microsoft Excel** — source dataset

## Data

The dataset (`Dataset PBI Playlist.xlsx`) is sample data used for visualization
and learning purposes.
